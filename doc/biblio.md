## <a name="headBB0"></a>The C Language Programming

* **Programming in C**. *Stephen G.Kochan* (`Addison Wesley, 2015, 4th`)

* **Expert C Programming - Deep C Secrets**. *Peter Van Der Linden* (`Prentice Hall, 1994`)
